This simple script serves to fetch all active tasks from Kanbanpad for one, a
few, or all users.

1. Copy all files to a location on your web server.
2. Copy default.config.php to config.php and edit config.php according to the
   inline documentation in that file.
3. Copy default.index.php to index.php, in a web-accessible location on your
   web server. Edit index.php according to the inline documentation in that file.
4. For additional users or alternate configurations, repeat step 3 as many
   times as necessary.
5. Point your browser to the directory used in step 3.
